Butcher's Delight: Rechopped is a spiritual successor to the mod [Butcher's Delight](https://www.curseforge.com/minecraft/mc-mods/butchers-delight). This mod features an entirely new way to manage meats, a full Create mod compatible butchering system, several new machines and tools, and many mechanics to dive into that make animals more interesting and realistic to manage.
There is a fabric version of this mod linked [here](https://github.com/Spicylemon2623/Butchers-Delight-Resown).

Planned Updates: <br>
    \- 1.0: Complete reimplementation of the original mod 1:1 <br>
    \- 1.X: Retexturing and remodeling of everything, bugfixing, and adding configs<br>
    \- 2.0: Compat with other mods and addition of a small addon api <br>
    \- 2.X: Make a seperate addon for [Create](https://github.com/Creators-of-Create/Create) compatibility <br>

Future Concepts (may be implemented when the base mod is finished): <br>

    \- More to come! <br>
